# WordPress-Elementor-Plugin-3.5.5-XSS-or-iFrame-Injection-POC
Proof of concept script for XSS or iFrame injection on Elementor Plugin for WordPress Versions &lt;=3.5.5

Usage: exploit.py

This script will first check to see if the site is vulnerable, then ask whether you want to use an XSS or iFrame Injection payload. You will then be able to customize the payload details, then it will give you the payload URL.
